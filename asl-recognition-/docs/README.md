# SIGN ASL Documentation

This folder contains the project documentation for SIGN ASL, a real-time American Sign Language recognition system built with React, Flask, MediaPipe, and TensorFlow.

- [project-report.md](project-report.md) - complete project report overview
- [structure.md](structure.md) - program flow, startup flow, and data flow

## Technical Documents

- [api.md](api.md) - backend API endpoints and example payloads
- [backend.md](backend.md) - backend modules, routes, models, and training flow
- [frontend.md](frontend.md) - frontend components, app layout, and state behavior
- [practice.md](practice.md) - practice mode rules, scoring, and UI elements
- [training.md](training.md) - data collection, sequence format, and word-model training
